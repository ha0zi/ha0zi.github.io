近况

2019-10-03

    8月23日和家中发生口角,出于不想受制于人的心态匆忙找了一份工作,在家中的外来企业步步高超市就职于信息主管岗位.很快面试通过,我进入了这家超市并匆忙的进行了工作.很快过了一个多月,我的工作很充实,但是在9月20号我忽然发现我已经很久没有学习了.以往朋友们有问题总是会咨询我,但现在我却发现自己已经无法跟上他们的话题了,这个发现我很惶恐,非常的不安!再加之工作内容的不断增加,我最终决定先结束这段不适合的工作,在处理其他因素的影响,但是出于情面我难以启齿.经过十多天的打气和不安之后最终于今日正式提交了辞职申请,过程比我想象的简单很多,我以为我的 BOSS 会恶言相向的.心中暗暗松了一口气.
    我不太喜欢和人说自己的实习经历,因为我认为我背负了这么好的学习机会,应该给予更优秀的人才得到这么好的实习经历,为医疗行业发光发热.我向往已一个不一样的身份走进以太网,和全世界的天才们互相交流发出思想的碰撞,我认为这很酷,我认为这样可以改变我的生活以及所有人的生活!我小时候写理想是成为一个依靠计算机生活的人,当时的我认为者很幸福;后来的我想成为 Hacker ,认为这很酷,但因为种种原因学习了 Java 并侥幸做了半年的码农;今日我拜读了<浪潮之巅>向往潮流引领者们的波澜壮阔,于是想成为他们一样的人.道路很难,时间很短,愿君当勉.与君共勉.

2019-08-07

俗话说好记性不如烂笔头，因此大部分人需要学习的话就需要做笔记记录。又因为是学习的目的，那么这个做笔记的服务只能自己书写了。有了笔记之后就是需要开始新的学习篇章，又说兴趣是最好的老师，那么我拥有了笔记的服务之后就可以根据自己的兴趣去学习记录了。但是我又想我的价值最大化，现阶段最热门的应该就是 AI 了，AI 很多人都对数学要求比较高，大部分文献又都是英文，很多人都认为 Python 是目前最好的实现语言。因此我的下一个目标是学习 Python。我需要前端技能，咨询过朋友之后前端技术目前选择 Vue 和 JS。

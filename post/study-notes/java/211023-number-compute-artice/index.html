  1. 当除数为 10 的 n 次幂;

提防精度丢失，可以考虑使用字符串分割然后拼接的方式获取结果

// 计算 432843 / 10^3
int a = 432843;

String aStr = String.valueOf(a);
Strint aLen = aStr.subtring(0, 3)
String aReg = aStr.subtring(3, 7);

return Integer.valueOf(aLen + "." + aReg);

立个 flag

Aug 7, 2019 | 闲谈

在我的概念里，立 flag 是个十分危险的事情。因为大概率会因为拖延症等原因而放弃，但是也不能完全没有目标。因此只能触触这个霉头了。

俗话说好记性不如烂笔头，因此大部分人需要学习的话就需要做笔记记录。又因为是学习的目的，那么这个做笔记的服务只能自己书写了。有了笔记之后就是需要开始新的学习篇章，又说兴趣是最好的老师，那么我拥有了笔记的服务之后就可以根据自己的兴趣去学习记录了。但是我又想我的价值最大化，现阶段最热门的应该就是 AI 了，AI 很多人都对数学要求比较高，大部分文献又都是英文，很多人都认为 Python 是目前最好的实现语言。因此我的下一个目标是学习 Python。我需要前端技能，咨询过朋友之后前端技术目前选择 Vue 和 JS。

目前迫于信用卡还贷的压力，在家里找了一份低工资的工作。这意味着我不需要明年开春必须找一份工作，也意味着我可以真正根据自己的兴趣进行学习。我可以看安全方面的书了！！！《黑客秘籍——渗透测试实用指南》走起～。～》》

那么我阶段性的安排是：

  • 使用 Java 实现一个用于记录笔记的博客系统。
  • 学习前端语言 vue 和 js。
  • 学习语言 Python
  • 期间穿插学习数学 和英语书写和阅读理解。
  • 计算机安全技术。

我写这个博客的原因

2019-07-02 | 闲谈

我搭建过博客的次数早就记不清了,至于为什么搭建那么多次…无非偷懒最后都文件都不见了.

为什么搭建这个博客

  • 想要看看书,认为自己想要有所作为就必须要看不少的书.

为什么选择 Hexo

  • 曾经考虑过相对成熟的 WordPress以及使用感觉很不错 Z-Blog最后还是选择使用 Hexo 主要基于以下几点考虑.

    • 资金: 目前无业,并且段时间也并不准备去寻找一份工作,所以需要尽可能的少花钱,如果选择之前两种博客进行部署,需要使用一台服务器;而使用 Hexo 可以使用 Github + Hexo 的形式,省云服务器费用.

    • 有一定经验: 曾经使用过,知道搭建和基本的使用方式,相对简单的操作不知道的也可以在对应的中文版官网上进行查阅

    • 储存: 无论选择 WordPress 亦或是 Z-Blog 都绕不开数据库的使用,数据库的搭建以及存储并且保证数据的持久化问题也是相对麻烦的.使用 Hexo 存储在 Github 是一方面,同时也会将整个文件存储在网盘中,真正需要保存的就是 /source/_posts 下的文件即可,相对来说需要存储的文件要小很多

    • 前面都是优势,缺点也有很多,对我来说最不舒服就是不方便,除了需要博文的编写之外还需要手动生成静态文件并且将它使用 Git 推送到远端,相对多了几个步骤;功能扩展方面的方便性也是不能比较的.
